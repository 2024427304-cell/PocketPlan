# 📱 PocketPlan

PocketPlan is a mobile expense tracking application developed using React Native and Firebase. The application helps users 
manage their monthly income, set budgets for different expense categories, record daily expenses, and monitor their spending 
habits through reports and spending summaries.

---

## ✨ Features

- User Registration & Login (Firebase Authentication)
- Monthly Income Setup
- Select Expense Categories
- Set Budget for Each Category
- Add New Expense
- Delete Expense
- Spending Overview
- Budget Progress by Category
- Spending Report
- Profile Management
- Edit Income
- Edit Budget
- Delete Selected Budget
- Logout

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/2024427304-cell/PocketPlan.git
```

Navigate to the project folder

```bash
cd PocketPlan
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npx expo start
```
